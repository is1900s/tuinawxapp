# 图标文件说明

## 底部导航栏图标

以下图标文件需要创建（建议尺寸：81x81px，PNG格式）：

### 首页图标
- `tab-home.png` - 未选中状态的首页图标（灰色）
- `tab-home-active.png` - 选中状态的首页图标（蓝色 #007AFF）

### 技师图标
- `tab-technician.png` - 未选中状态的技师图标（灰色）
- `tab-technician-active.png` - 选中状态的技师图标（蓝色 #007AFF）

### 订单图标
- `tab-order.png` - 未选中状态的订单图标（灰色）
- `tab-order-active.png` - 选中状态的订单图标（蓝色 #007AFF）

### 用户图标
- `tab-user.png` - 未选中状态的用户图标（灰色）
- `tab-user-active.png` - 选中状态的用户图标（蓝色 #007AFF）

## 设计规范

### 尺寸要求
- **图标尺寸**: 81x81px (微信小程序tabBar标准尺寸)
- **格式**: PNG (支持透明背景)
- **颜色深度**: 32位

### 色彩规范
- **未选中状态**: #8E8E93 (灰色)
- **选中状态**: #007AFF (微信蓝)
- **背景**: 透明

### 设计风格
- 简洁、现代的扁平化设计
- 线条粗细适中，确保在小尺寸下清晰可见
- 保持图标风格统一

### 建议图标样式

**首页 (🏠)**
- 房子轮廓线条图标
- 简洁的屋顶和墙体结构

**技师 (👨‍💼)**
- 人物轮廓或服务相关的图标
- 可以使用简化的人物形象或按摩相关元素

**订单 (📋)**
- 文档或列表图标
- 简化的纸张或清单样式

**用户 (👤)**
- 人物轮廓图标
- 简洁的用户头像形状

## 临时解决方案

如果您暂时没有图标文件，可以：

1. **使用字体图标**: 修改app.json，移除tabBar配置，使用自定义tabBar组件
2. **使用网络图标**: 将图标上传到CDN，使用网络地址（需要配置合法域名）
3. **使用emoji**: 在自定义tabBar中使用emoji作为临时图标
